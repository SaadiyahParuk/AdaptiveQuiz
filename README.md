# *(Master Branch Contains The Android Project)*
# AdaptiveQuiz - LearnQuest
Android Studio App Based on My AdaptiveGamifiedLearningSystem Machine-Learning Model

# Overview
This mobile application implements a machine learning-based CRUD system that predicts student performance and adapts learning content accordingly. The system integrates AI, gamification, and is native to android.

# Specifications used
# Polyglot Mobile Application
- Kotlin
- Java

# CRUD
- Users are able to create, play, edit and delete quiz sets

# System
- Android studio (Panda 1 | 2025.3.1 Patch 1)
- Kernel (Built in)

# Application
- Jetpack Compose
- Material 3
- .xml Layout
- Retrofit
- OkHttp
- Room
- Firebase
- SQLite
- Google Services (Google Cloud Platform)
- FireAuth

# Networking
- Retrofit interface, request and response data classes
- Singleton network object that builds an OkHttpClient

# Security
- FireAuth

# Emerging Technologies - Machine Learning
- Implemented my API
- Features used:
  - studytime (timeElapsed)
  - failures, absences (no. of incorrect answers)
  - G1 (current score)
  - G2 (previous score)
- Target:
  - G3 (binarized)
